## Project Installation
Follow the steps properly -

- Flutter SDK Must be installed properly.

- Clone Repository
```bash
  gh repo clone vaibhavkusalkar/Food_Delivery
```

- Get into the Project folder (i.e Food_Delivery)
```bash
  cd Food_Delivery
```

- Run
```bash
  flutter clean
```
```bash
  flutter pub get
```

The App is ready to use !
